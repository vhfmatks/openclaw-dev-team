# OpenClaw 개요

## 정의

**OpenClaw** (구 Clawdbot, Moltbot)는 2025년 11월에 출시된 오픈소스 자율형 AI 에이전트입니다.

## 핵심 특징

### 1. 빠른 성장
- GitHub 스타 180,000+ 개
- 사용자 300,000~400,000명 (출시 90일 만에)
- 역사상 가장 빠르게 성장하는 오픈소스 AI 에이전트

### 2. 자율형 에이전트 (Agentic AI)
- 단순 질문 응답이 아닌 **사용자를 대신해 행동**하는 AI
- 이메일 연동, 회의 예약, 메시지 작성, 피드 모니터링
- 독립적인 의사결정 및 작업 수행

### 3. 주요 사용 사례

#### 변호사/법률 분야
| 기능 | 설명 |
|------|------|
| 문서 검토 | 계약서, 사건 파일, 디스커버리 문서 자동 분석 |
| 클라이언트 커뮤니케이션 | 고객 응대, 상태 업데이트 이메일 자동화 |
| 기한 추적 | 법정 제출 기한, 미팅 일정 관리 |
| 청구 시간 최적화 | 주당 15시간 절약 (행정 업무 감소) |

#### 의료/금융 분야
- 의료 기록 분석
- 금융 문서 처리
- 규정 준수 모니터링

## 보안/법적 고려사항

### 리스크
1. **루트 액세스 권한** - 사용자 컴퓨터에 깊은 접근 권한
2. **확장 프로그램 취약점** - 4개 중 1개가 보안 취약점 포함
3. **자격 증명 탈취 위험** - 일부 악성 확장 프로그램 존재
4. **법적 책임** - AI의 자율적 행동에 대한 책임 규정 미명확

### 기업 도입 시 고려사항
- 거버넌스 정책 수립 필수
- 보안 감사 필요
- 데이터 처리 방침 검토
